# Мои учебные проекты на C#

Здесь я изучаю основы программирования на C# и практикуюсь в написании консольных приложений 
с обработкой пользовательского ввода, циклами, условиями и современными конструкциями языка (switch-выражения).

Все программы устойчивы к некорректному вводу: они не падают, если пользователь ввёл буквы вместо чисел
или выбрал неверный вариант — вместо этого выводят понятное сообщение об ошибке и предлагают повторить ввод.

---

## 📁 Список проектов

### 1. If-Else Examples (`if-else-examples.zip`)
**Файл:** `Инструкция else.cs`  
**Namespace:** `IfElseExamples`

Примеры использования условных операторов `if/else`:
- Проверка уровня игрока (≥100 — пройден)
- Приветствие по времени суток (утро/день/вечер)
- Интерактивный светофор (red/yellow/green)

✅ Особенности:
- Обработка нечислового ввода через `int.TryParse`
- Валидация диапазона часов (0–23)
- Чёткая структура методов с комментариями

---

### 2. Switch Expressions (`switch-expressions.zip`)
**Файл:** `Program.cs`  
**Namespace:** `Инструкция_Switch`

Демонстрация современных switch-выражений (C# 9):
- Выбор категории (Sport/Business/Technology) по номеру
- Выбор цвета (red/blue) по строке

✅ Особенности:
- Использование кортежей `(bool, int)` для безопасной обработки ввода
- Паттерн `_` для обработки ошибок
- Лаконичный синтаксис без лишних `break`

---

### 3. Feet-Inches Converter (`feet-inches-converter.zip`)
**Файл:** `Program.cs`  
**Namespace:** `FeetInchesConverter`

Конвертер единиц измерения с циклическим режимом работы:
- Футы → Дюймы
- Дюймы → Футы
- Возможность продолжить работу после каждого расчёта

✅ Особенности:
- Цикл `while` с возможностью выхода по команде "нет"
- Обработка нечислового ввода через `double.TryParse`
- Разделение логики на методы: меню, конвертация, запрос продолжения

---

### 4. Simple Calculator (`simple-calculator.zip`)
**Файл:** `Program.cs`  
**Namespace:** `SimpleCalculator`

Простой калькулятор с поддержкой базовых операций:
- Сложение (+)
- Вычитание (-)
- Умножение (*)
- Деление (/)

✅ Особенности:
- Защита от деления на ноль
- Циклический режим работы
- Валидация ввода чисел и выбора операции
- Использование switch-выражений для маппинга операций

## ▶️ Как запустить проект

1. Скачайте нужный ZIP-архив из репозитория.
2. Распакуйте архив в любую папку.
3. Откройте папку в **Visual Studio** или **VS Code**.
4. Запустите файл `Program.cs` (нажми F5 или используй терминал: `dotnet run`).
5. Следуйте инструкциям в консоли.

> 💡 Все проекты работают в консольном режиме и не требуют установки дополнительных библиотек.
